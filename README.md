# INTRODUCTION:

This dashboard analyzes salary trends and job demand within the data profession.
It highlights the highest paying data roles based on median yearly salary.
The report also compares salary variations across countries such as the United States, United Kingdom, France, and India.

Additionally, it examines the frequency of job postings that do not mention a degree requirement.
Overall, the visualizations provide insights into compensation patterns and education requirements in the global data job market.

In this dashboard we used different Column and Bar charts like 
1) Stacked bar chart 
2) Stacked column chart 
3) Clustered column chart 
4) 100% stacked bar chart

# INTRODUCTION:

This dashboard provides a comprehensive overview of data job market trends in 2024.
It analyzes monthly job posting trends to identify fluctuations and overall demand patterns throughout the year.
The report compares different data roles, including Data Analyst, Data Engineer, Data Scientist, and senior level positions.
It also examines the percentage share of each role to understand how job distribution changes over time.
Additionally, the dashboard highlights salary comparisons between yearly median salary and hourly median pay.
Overall, the visualizations offer insights into job demand, role distribution, and compensation trends in the data industry.

In this dashboard we used different Line and Area charts like 

1) Line chart
2) Multi-Line chart
3) Stacked Area chart
4) Clustered column chart with line
