# Column & Bar _Visualization/ Power Bi

In this project I build different Column and Bar charts like 
1) stacked bar chart 
2) stacked column chart 
3) clustered column chart 
4) 100% stacked bar chart

## Column Chart:
Column charts are helpful in comparing values across categories,Showing trends over time (months, years).

## Bar Chart:
Bar charts are helpful in comparing many categories,also when the categories names are long.
